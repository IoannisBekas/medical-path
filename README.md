# Medical Path Website

Greek-first static website for Medical Path, a medical supply and equipment company in Glyfada, Athens.

## Live Site

This repository is intended to be published with GitHub Pages from the repository root.

## Local Development

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

Then open `http://127.0.0.1:8765/`.

## Structure

- `index.html` - Greek homepage
- `styles.css` - responsive styling
- `assets/` - local location graphic and favicon
- `robots.txt` - crawler rules
- `sitemap.xml` - sitemap for the published GitHub Pages URL
- `IMPROVEMENT_PLAN_EL.md` - detailed improvement plan
